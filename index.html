What is it Scripting Windows Host Console?


Scripting Windows Host Console is a program that allows you to have more control on your system via command-line

You can do lots of things, like create files, folders, delete files, folders, change system time and date, end processes, check if a process is running, encode files in Base64, encrypt text with a private key, run files, run programs as administrator, and more!
